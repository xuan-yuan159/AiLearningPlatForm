# 部署指南

## 1. 构建与产物

```bash
cd Frontend/OnlineLearning
npm install
npm run build
```

构建产物位于 `Frontend/OnlineLearning/dist`。

## 2. Nginx 配置示例

```nginx
server {
    listen 80;
    server_name teacher-admin.example.com;
    root /var/www/online-learning/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api/ {
        proxy_pass http://backend-service:8080/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

## 3. 环境变量示例

```bash
VITE_API_BASE_URL=https://api.example.com
VITE_APP_TITLE=AI助教教师后台
VITE_ENABLE_MOCK=false
```

## 4. CI/CD 脚本示例（GitHub Actions）

```yaml
name: frontend-ci
on:
  push:
    branches: [main, develop]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm
          cache-dependency-path: Frontend/OnlineLearning/package-lock.json
      - name: Install
        run: npm ci
        working-directory: Frontend/OnlineLearning
      - name: Lint
        run: npm run lint
        working-directory: Frontend/OnlineLearning
      - name: Test
        run: npm run test:coverage
        working-directory: Frontend/OnlineLearning
      - name: Build
        run: npm run build
        working-directory: Frontend/OnlineLearning
```

## 5. 测试报告生成命令

```bash
npm run test:coverage
```

报告目录：`Frontend/OnlineLearning/coverage/`。
