# 教师后台使用说明

## 1. 系统入口

- 本地开发地址：`http://localhost:5173/login`
- 默认演示账号：`teacher@ailearning.com`
- 默认演示密码：`123456`

## 2. 页面与路由总览

| 页面         | URL               | 主要功能                   | 关键状态           |
| ------------ | ----------------- | -------------------------- | ------------------ |
| 登录         | `/login`          | 教师账号登录、登录失败提示 | 加载、错误         |
| 仪表盘       | `/dashboard`      | 指标概览、待办提醒         | 加载、空数据       |
| 课程管理     | `/courses`        | 课程列表、课程创建入口     | 加载、空数据、错误 |
| 课程资源建设 | `/resources`      | 课程章节资源概览、视频上传 | 加载、空数据、错误 |
| 在线问答管理 | `/qa`             | 问答列表、AI 草稿与发布    | 加载、空数据、错误 |
| AI 摘要审核  | `/ai-summaries`   | 摘要任务、版本对比与发布   | 加载、空数据、错误 |
| 学生管理     | `/students`       | 学生进度与风险预警         | 空数据             |
| 通知公告     | `/notices`        | 公告发布、阅读率查看       | 空数据             |
| 个人中心     | `/profile`        | 教师资料维护               | 错误               |
| 无权限       | `/no-permission`  | 无权限访问提示             | 无权限             |
| 低保真原型   | `/prototype/low`  | 信息架构与交互流程草图     | 加载               |
| 高保真视觉稿 | `/prototype/high` | 视觉规范与组件层级         | 加载               |

## 3. 页面组件结构

- 登录页：`LoginView` + 表单输入 + 登录按钮 + 错误提示
- 后台布局：`AdminLayout`（侧边菜单 + 顶部标题 + 子路由视图）
- 仪表盘：指标卡片网格 + `PageSection`
- 表格类页面：`PageSection` + `DataTable`
- 状态显示：`StatusTag` 统一渲染发布状态、预警状态

## 4. 状态管理方案

- 鉴权状态：`src/stores/useAuthStore.js`
  - 管理 token、用户资料、登录退出、资料更新
  - 数据持久化到 LocalStorage
- 后台业务状态：`src/stores/useAdminStore.js`
  - 管理仪表盘指标、全局 loading、学期筛选条件
- 数据请求层：`src/api/httpClient.js` + `authApi.js` + `teacherApi.js`
  - 提供统一异常抛出与 mock 数据响应

## 5. 关键交互状态说明

- 加载态：表格组件显示“加载中...”
- 空数据态：表格组件显示“暂无数据”
- 错误态：登录页显示错误文案，请求层抛出异常后由页面接管
- 无权限态：访问控制失败后跳转 `/no-permission`
